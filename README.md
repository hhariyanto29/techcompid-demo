# Discovery Kuta Festival — Prototype

Static HTML/CSS/JS prototype for the Discovery Kuta Festival microsite (Aug 2026 – Jan 2027).

## Pages

- `index.html` — landing
- `signup.html` — member sign up
- `member.html` — member dashboard
- `tenant.html` — merchant view
- `scanner.html` — staff scanner
- `admin.html` and `admin-*.html` — admin area (events, members, merchants, points, redemptions, reports, rewards, roles, settlements, transactions)
- `flow-admin.html`, `flow-member.html`, `flow-merchant.html` — UX flows

## Run locally

Open `index.html` directly, or serve the folder:

```
python3 -m http.server 8000
```

Then visit http://localhost:8000.
